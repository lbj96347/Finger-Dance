Finger Dance
===================

手指舞

是2012年3月24、25日在HTMl5研究小组以及中青宝联合举办的"HTML5 Code Jam"中诞生的作品。

作品由：碧青，小虎，小唯，仁柱，秉骏完成。

历时32小时。

作品引用的第三方框架以及库：

  1,magic_touch.js 方便捕获多点触控的时候，点在屏幕的位置；

  2,jqmobi 选择器的库，专为移动设备重写的jquery;

  3,add2home.js插件，告知用户“添加到主屏幕”以安装该应用。
